### Commands ###
- /도와주새우 : 도움말 명령어를 표시합니다.
- /뉴스검색 <키워드> : 입력한 키워드로 뉴스를 검색합니다. 정확도가 반영된 100개의 최신 뉴스를 가져옵니다.
  > 선택메뉴를 통해 보고싶은 기사를 선택할 수 있습니다. 제목을 클릭하면 기사 전문으로 이동합니다.                    
  > <(이전페이지), >(다음페이지) 버튼으로 페이지를 이동할 수 있습니다.                         
  > X(검색종료) 버튼을 누르면 검색창을 닫을 수 있습니다.
